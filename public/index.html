Como usar:

Utilize o método GET para obter um ou mais registros, filtrados pela URL.

Exemplos:

GET /produtos (obtém uma lista com todos os produtos)

GET /produtos?marca=loreal (obtém uma lista de produtos da marca Loreal)

GET /produtos?nome=shampoo&marca=loreal (obtém uma lista de shampoos da marca loreal)

GET /produtos?_id=123123123 (obtém o produto com id=123123123)

Como usar:

Utilize o método POST para adicionar um registro.

Corpo da requisição:

Envie o corpo da requisição POST como um objeto JSON, contendo pelo menos os itens obrigatórios do objeto (consulte os Modelos).

Resposta:

Se o registro for adicionado com sucesso, a resposta será o próprio registro adicionado, mais o ID (campo _id) gerado pelo banco. Caso contrário, a resposta será um objeto contendo um erro.
